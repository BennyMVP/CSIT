School project focused on biomedical image segmentation using different image processing techniques in MATLAB.

This project explores and compares several segmentation methods applied to biomedical images, including brain MRI and chest CT scans.
The work includes classical image processing techniques such as thresholding, edge detection, region growing, K-means clustering, active contours, and a hybrid approach that combines region growing with active contours.

The goal of the project is to analyze how different segmentation techniques behave on medical images and evaluate their results using quantitative metrics such as Dice coefficient, IoU, Precision, Recall, HD95, and execution time.

The project was developed in MATLAB as part of an academic course.
